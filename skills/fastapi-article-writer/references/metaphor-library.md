# 导读比喻库

撰写导读时，从以下比喻类型中选择最贴切的场景。

## 1. 餐厅/外卖类（最常用，优先级高）

- 异常处理 = 餐厅应对挑剔客人、处理投诉
- 文件上传 = 外卖配送服务、餐品打包
- 依赖注入 = 餐厅备料、厨师按菜单取食材
- 中间件 = 餐厅前台接待、叫号系统

## 2. 交通/高速公路类

- 中间件 = 高速公路收费站、检查站
- 性能优化 = 车辆加速、车道扩容
- 路由分发 = 交通导航、路牌指引
- 请求响应 = 快递运输流程

## 3. 停车场/车位管理类

- 连接池 = 停车位管理、车位预约
- 资源限制 = 停车场满员控制
- 并发处理 = 多车道进出

## 4. 工作/办公场景类

- Webhook = 自动化构建触发器、消息通知
- 分布式锁 = 双11秒杀库存控制
- 任务队列 = 工厂流水线
- 定时任务 = 闹钟提醒、日程表
- AI Prompt = 给助手下指令、写工作简报
- AI 模型调用 = 请教专家、外包复杂任务

## 5. 生活日常类

- 缓存 = 便利店 vs 大超市
- 认证授权 = 门禁卡、身份证验证
- 版本控制 = 软件更新提示
